Help !

Thank you for using our website during this closed beta phase.

- You can browse the games library using filters, or search
- Just type to start searching
- Downloads are not 'direct downloads', you pick a game and queue it for downloading. The server will prepare a ZIP file for downloading.


ADMIN TIPS :

- Use auto folder scanner to scan your games folders
- Folder scanning will take all subfolders of given path and try to add each of them as games to the library
- Do not start second job before first one is done
- You can add any folder your server has access to
- Game names are automatically cleaned
- Check unmatched folder after scanning is done
